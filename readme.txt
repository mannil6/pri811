
Для запуска приложения необходима установка java 1.8 на компьютере.
Запуск приложения производится при помощи файла run.bat
Файл run.bat должен находиться в папке, где находится файл приложения area-1.jar
-------------------------------------------------------------------------------
Внимание : Перед запуском приложения необходимо проверить права у приложения ,
добавить его в исключения антивирусу



Сборка приложения
-------------------------------------------------------------------------------
Для сборки приложения на компьютере должен быть установлен maven
Собирается приложение командой  mvn clean install
